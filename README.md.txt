# PP_StockVerse AI

### Learn • Analyze • Practice • Grow

PP_StockVerse AI is an AI-powered Stock Learning and Paper Trading Platform designed for beginners and aspiring investors.

The platform helps users learn stock market concepts, practice trading with virtual money, analyze portfolios, and receive AI-driven insights.

---

## Features

* User Authentication
* Market Dashboard
* Stock Search
* Paper Trading
* Portfolio Management
* AI Mentor
* AI News Summary
* Learning Feed
* Quiz & XP System
* Community Discussion
* Leaderboard
* Admin Panel
